
# AI Life OS

AI Life OS is an AI-powered email management and productivity assistant built using Make.com, Groq Llama 3, Gmail, Google Sheets, and HTTP APIs.

## Features

* Monitors incoming Gmail emails
* Uses Llama 3 (Groq API) to classify emails
* Categories:

  * Placement
  * College
  * Personal
  * Important
* Stores classified emails in Google Sheets
* Generates automated daily summaries
* Sends daily reports through Gmail

## Tech Stack

* Make.com
* Groq API (Llama 3)
* Gmail
* Google Sheets
* HTTP APIs

## Workflow

Gmail → Groq AI → Classification → Google Sheets

Scheduler → Google Sheets → Groq AI → Gmail Report
